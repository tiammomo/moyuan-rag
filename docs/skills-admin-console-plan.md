# Skills Admin Console Plan

## Goal

Surface the new governance data in the frontend so operators can review install tasks, audit logs, and version decisions without direct API inspection.

## Checklist

- [pending] Add an admin-only skills governance page for install tasks and audit logs.
- [pending] Add filters for task status, action type, actor, skill slug, and robot ID.
- [pending] Add detail drawers for install task payloads and audit log JSON details.
- [pending] Add version comparison UI for installed skill variants and pinned bindings.
- [pending] Add operator actions for safe rebind and rollback preparation workflows.
