# Skills Audit Plan

## Goal

Audit whether the current project has a complete front-to-back "skills" workflow, identify unfinished frontend buttons or missing backend support, and evaluate whether remote skills can be pulled down to local storage directly.

## Checklist

- [pending] Audit the frontend for `skills` entry points, buttons, and interaction flows.
- [pending] Audit the backend for `skills` APIs, services, storage, and remote-download capabilities.
- [pending] Cross-check frontend actions against backend support and list incomplete or unwired buttons.
- [pending] Decide whether remote skills download should be implemented directly in this repo or delegated to an external installer workflow.
- [pending] Sync `README.md` and docs with the audit findings and any agreed implementation plan.
