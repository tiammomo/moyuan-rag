# Skills Marketplace Hardening Plan

## Goal

Prepare the `skills` capability for a future controlled marketplace phase without enabling remote install by default yet.

## Checklist

- [pending] Add signed-package and checksum verification requirements for future remote skill packages.
- [pending] Add install task status persistence so remote and local installs can be audited.
- [pending] Add admin-only audit logs for skill install, bind, update, and unbind operations.
- [pending] Define rollback and version pinning behavior for skill upgrades.
- [pending] Design a constrained remote source allowlist model and operator runbook.
